### 1.1 环境配置

| 配置项 | 开发环境 | 测试环境 | 生产环境 |
|--------|---------|---------|---------|
| 基础 URL | `http://localhost:8080` | `http://test-api.example.com` | `https://api.example.com` |
| 数据库 | MySQL 8.0+ | MySQL 8.0+ | MySQL 8.0+ |
| Redis | Redis 6.0+ | Redis 6.0+ | Redis 6.0+ |

### 1.2 测试工具

- Postman / Apifox
- JMeter（性能测试）
- curl（命令行测试）

### 1.3 通用请求头

| Header | 说明 | 示例 |
|--------|------|------|
| `Content-Type` | 请求内容类型 | `application/json` |
| `Authorization` | Token（除登录接口外） | `Bearer {token}` |

### 1.4 统一响应格式

```json
{
  "code": 0,
  "msg": "success",
  "data": {}
}
```
